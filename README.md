# CIFAR-10

Classification of CIFAR-10 dataset using a Deep Convolutional Neural Network.

# Tools
1. Visual Studio Code
2. Python 3.8
3. TensorFlow 2.x

# Results

### TRAINING (50,000 examples)

1. Accuracy - 87.17%
2. Loss - 0.3812

### VALIDATION (5,000 examples)

1. Accuracy - 77.20%
2. Loss -  0.7815

### TESTING (5,000 examples)

1. Accuracy - 76.80%
2. Loss - 0.808
